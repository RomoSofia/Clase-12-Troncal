# Acreditación, arancel y oferta de carreras de Diseño en Chile

Este proyecto corresponde a una página web desarrollada en HTML, CSS y JavaScript para visualizar datos asociados a carreras de Diseño en el sistema de educación superior chileno.

El trabajo analiza carreras de Diseño en Chile a partir de datos del Servicio de Información de Educación Superior (SIES), considerando principalmente universidades, institutos profesionales y centros de formación técnica.

La página responde dos preguntas mediante visualizaciones:

1. **¿Existe una correlación directa entre el nivel de acreditación de una universidad y el costo de su arancel anual?**

2. **¿Qué tipo de institución concentra más vacantes para estudiar Diseño en 2026?**


## Visualizaciones utilizadas

### Gráfico de dispersión

Se utiliza un gráfico de dispersión para observar la relación entre dos variables numéricas:

* Años de acreditación institucional.
* Arancel anual del programa.

Este tipo de gráfico permite identificar si existe una tendencia o correlación entre ambas variables.

### Gráfico de dona

Se utiliza un gráfico de dona para representar una relación de parte a todo.

En este caso, el gráfico muestra cómo se distribuyen las vacantes totales 2026 de carreras asociadas a Diseño entre:

* Universidades.
* Institutos Profesionales.
* Centros de Formación Técnica.

Las vacantes totales fueron calculadas sumando las vacantes del primer y segundo semestre.

## Archivos del proyecto

El proyecto está compuesto por tres archivos principales:

* `index.html`: contiene la estructura del sitio y el contenido textual.
* `style.css`: contiene los estilos visuales de la página.
* `script.js`: contiene la programación de los gráficos con Chart.js.

## Fuentes de datos

* Servicio de Información de Educación Superior (SIES), bases de datos de oferta académica 2026.
* Comisión Nacional de Acreditación (CNA), información de acreditación institucional.
* Bases consolidadas de carreras de Diseño en universidades, institutos profesionales y centros de formación técnica.
* Catálogo de Visualización de Datos, usado como referencia para justificar la elección de los tipos de gráficos.

## Cómo visualizar el proyecto

Para abrir el proyecto, se deben mantener los archivos `index.html`, `style.css` y `script.js` en una misma carpeta.

Luego, basta con abrir el archivo `index.html` en un navegador web.
